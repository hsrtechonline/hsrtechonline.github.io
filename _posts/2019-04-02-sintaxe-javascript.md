# Sintaxe Javascript

# Variáveis
*Variáveis JavaScript são contêineres para armazenar valores de dados.*

No exemplo abaixo x, y e z são variáveis. x recebe 5, y recebe 6 e z recebe a soma de x e y.  
<pre>
<code>var x = 5;
var y = 6;
var z = x + y;
Resultado = 11</code>
</pre>
